# 控制工程基础知识要点

## 简答(25 分)

1. 自动控制原理定义、特征
2. 开环系统与闭环系统
3. 闭环系统构成
4. 自动控制系统基本要求
5. 非线性系统如何实现线性化
6. 传递函数定义、性质
7. 常见环节传递函数
8. 控制系统稳态过程概念、控制系统瞬态过程概念
9. 系统稳定条件
10. 阻尼系数对二阶系统瞬态响应性能影响
11. 主导极点概念
12. PID 原理中 P、I、D 作用

## 计算(7 小题, 75 分)

1. 根据物理模型列微分方程，并求其传递函数
2. 结构图的化简
3. 信号流图及梅逊公式
4. 稳定裕度、劳斯判据
5. 高阶系统低阶化
6. 一阶、二阶系统瞬态响应的性能指标
7. 带噪声稳定系统给定传递函数求稳态误差
