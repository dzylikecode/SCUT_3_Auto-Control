## 定义

自动控制是指在**无人直接参与**的情况下，通过**控制器**使**被控对象或过程自动地按照预定**要求进行

- 控制器

- 被控对象

- 自动控制系统

  控制器与被控对象的组合

- 闭环系统中各基本环节的作用

- 开环与闭环系统的特点

  - 开环

    顺向作用，没有反向的联系，没有修正偏差能力

  - 闭环

    偏差控制，可以抑制内、外扰动对被控制量产生的影响
